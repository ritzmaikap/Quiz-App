## Quiz App

The Quiz App is an interactive Android application designed to test user knowledge through multiple-choice questions. The app provides a smooth and engaging quiz experience by allowing users to select answers, receive instant feedback, and track their progress in real time.

Each question includes multiple options, where the correct answer is highlighted in green and incorrect selections are shown in red after submission. The app prevents users from changing their answers once submitted, ensuring fair evaluation. A progress bar is used to visually indicate the user’s advancement through the quiz.

The application also includes a result screen that displays the user’s final score upon completion of all questions. Additionally, a Dark Mode and Light Mode toggle is implemented, allowing users to switch themes dynamically, with preferences stored using SharedPreferences for consistency across sessions.

This project demonstrates key Android concepts such as activity navigation, UI interaction handling, real-time feedback, and theme management.
